#자료형 (문자형) : 네이버 블로그







아스키 코드로 문자를 표현할 수 있다

A = 65

B = 66

​

a = 97

b = 98

​

대문자를 입력받아 소문자를 출력

 char code;

scanf("%c", &code);

printf("%c", code + 32); 

​

scanf\_s시 ? 에러가 날때는 scanf\_s char 같은 키워드로 검색하자

​

아스키 코드는 printf에서 %d 사용시 정수형으로 표헌할 수 있다

65 = A

​

제어 문자

제어문자는 \로 시작한다

\n 줄바꿈

\a 경고

\b 백스페이스

등등 다양한 것들이 있으니 검색해서 잘 활용하자

​

\b의 활용법으로는 문자가 입력되있는상태에서 다시 입력시킬 수 있다

예시)

printf 로 \*\*\*\* 출력

\b\b\b\b //백스페이스 네번

scanf 입력

(결과)

\*\*\*\* → (a입력) → a\*\*\* (b입력) → ab\*\* → (cc입력)→ abcc
